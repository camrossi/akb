NKT Supports also baremetal nodes, however the configurations steps are sligtly more complex. When deploying on baremetal follow these steps
  • The BM nodes must be pre-configured with:
    • Host name: This will have to match the host name passed to NKT in teh Cluster definition page
    • Physcal interfaces: needs to be pre-condifured in the Cluster Subnet and use the ACI Floating SVI as default gateway
    • Physcal interfaces: Must use VLAN tagging, the rationale is that you could re-use the same physical interface also for in-band access/management
    • IP address: This will have to match the IP address passed to NKT in the Cluster definition page
    • (Optional) Interface boinding
  • On ACI A VLAN must be allocated to carry the traffic generated by the nodes
